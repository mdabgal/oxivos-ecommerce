# Festive Premium E-Commerce Store

Welcome to , a modern and responsive e-commerce front-end application crafted for premium traditional and casual clothing brands. This project showcases a high-end UI layout featuring modern Panjabis, designer Kurtis, and premium shirts with pixel-perfect responsive designs and rich animations.

##  Key Features

- ** Dynamic Dark & Light Mode:** Seamless transition-colors state management for absolute visual comfort.
- **Framer Motion Animations:** Ultra-smooth fade-ins, staggered listings, and interactive hover effects.
- ** Next.js Architecture:** Built on top of Next.js 16 (App Router) with fast rendering using Turbopack compiler.
- ** Tailwind CSS:** Clean, scannable, component-driven styles utilizing utility-first modern approaches.
- ** Grid Layouts:** An advanced `CategoryGrid` structure emphasizing major seasonal drops like our Premium Shirt line.

---

## Tech Stack

- **Framework:** Next.js 16 (React 19)
- **Compiler:** Turbopack
- **Styling:** Tailwind CSS
- **Motion:** Framer Motion
